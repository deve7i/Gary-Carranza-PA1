HTML, CSS y JavaScript:

Son los tres pilares fundamentales del desarrollo web que trabajan juntos para crear páginas web dinámicas e interactivas. HTML se encarga de la estructura y el contenido, CSS del diseño y la presentación, y JavaScript de la interatividad y la funcionaliada.

Definiciones:

HTML (HyperText Markup languaje):

Es el lenguaje de marcado que define la estructura y el contenido de una página wweb.Utiliza etiquetas para delimitar diferentes elementos como párrafod, encabezados, imágenes, enlaces, etc.

CSS (Cascading Style Sheets):

Es el lenguaje de estilos que define la presentación de una página web. Permite definir aspectos como los colores, fuentes, tamaños,márgenes, etc, de los elementos HTML.

JavaScript (Program Languaje):

Es el lenguaje de programación que define la interactiva auna página web.Permite manipular elementos HTML, responder a eventos del usuario realixar cálculos etc.

Analogía de la web y una casa:

HTML

Estructura de la casa.

Es como los cimientos, las paredes, techos, y distribución de las habitaciones.

HTML define la estructura básica de una página web: qué elementos hay(titulo, párrafos, botones, imágenes, etc), pero sin preocuparse por cómo se ven

CSS

Decoración y estilo de la casa.

Es la pintura de las paredes, el tipo de piso, los muebles, las cortinas, el diseño de la cocina.

CSS se encarga de cómo se ve la página: colores, fuentes, máegemes, alineaciones, estilos responsitivos... los visual.

JavaScript

Sistemas eléctricos y electronicos de la casa.

Es como los intercorruptores que encienden luces, las puertas autoamticas, el termostato inteligente.

JavaScript añade funcionalidad e interactividad haciendo que pasen cosas como respuesat aun evento.

Resumen

"Los tres lenguajes son esenciales para el desarrollo web, y cada uno juega en papel crucial en la creación de paginas web modernas y funcionales"
